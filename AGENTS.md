# Agent skills in this repository

Human and agent readers: executable workflows live under `skills/`. Each subdirectory contains a `SKILL.md` file with YAML frontmatter (`name`, `description`) and step-by-step instructions.

Start with the [README](README.md) skill index, then open the relevant `skills/<name>/SKILL.md`. For installs via the [skills.sh](https://skills.sh/) CLI, see [docs/skills-sh.md](docs/skills-sh.md).
